How to install the tyler oldham mod for SuperTux
1) Download the mod
2) press windows+R
3) In the window that appears type in "%appdata%" and hit enter or ok
4) go to supertuxkart, addons, karts
5*) if there isnt a folder called "karts", make one
6) unzip the file into there, it should be a folder called "tyleroldham" now
it should look something like %appdata%\supertuxkart\addons\karts\tyleroldham\ kart.mxl, materials.xml, catnonce.png, etc
